🧠 Proyecto: Corrección de Código Java - Tarifas de Programadores
📝 Descripción
Este proyecto es un programa en Java que solicita información de programadores (nombre, edad, experiencia, tarifas, etc.) y realiza cálculos para obtener valores como el total a pagar. Durante la revisión del código se detectaron y solucionaron varios errores importantes.

🔧 Cambios Realizados
✅ 1. Corrección del Scanner

Se arregló la declaración incorrecta del objeto Scanner.

Se cambió scaner por sc para evitar errores de compilación.

💰 2. Arreglo en la Tarifa Base

Se eliminaron comas incorrectas en el valor de la tarifa base que generaban errores.

🔤 3. Corrección de Nombres de Variables

Algunas variables estaban mal escritas (horasproyecto1, etc.), lo cual impedía que el código las reconociera.

Se ajustaron para que coincidan con sus declaraciones.

🙈 4. Variables Comentadas sin Leer

Se encontraron líneas de código comentadas que impedían la lectura de variables necesarias.

Fueron descomentadas para garantizar su ejecución.

💥 5. Error en el Subtotal

El cálculo del subtotal tenía errores como ==== que fueron removidos para evitar fallos de sintaxis.

🧹 6. Limpieza General del Código

Se revisó la coherencia entre variables y sus usos.
